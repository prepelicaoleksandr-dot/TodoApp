# 📝 Todo App - Управление Задачами

Современное веб-приложение для управления задачами, созданное на React. Идеально подходит для портфолио frontend разработчика.

## ✨ Особенности

- ✅ **Добавление задач** - Создавайте задачи с названием и опциональным описанием
- 📊 **Фильтрация** - Просмотр всех задач, только активных или выполненных
- 💾 **Сохранение** - Все данные автоматически сохраняются в localStorage
- 🎨 **Красивый дизайн** - Современный UI с плавными анимациями
- 📱 **Адаптивность** - Полная поддержка мобильных устройств
- ♿ **Доступность** - ARIA labels и поддержка клавиатурной навигации
- 🚀 **Высокая производительность** - Оптимизированный React код

## 🛠️ Технологический Стек

- **React 18.2.0** - Библиотека для создания интерфейсов
- **SCSS** - Препроцессор CSS для удобной работы со стилями
- **React Scripts** - Инструмент для создания и запуска React приложений
- **LocalStorage API** - Сохранение данных на клиентской стороне

## 🚀 Быстрый Старт

### Предварительные требования

Убедитесь, что у вас установлены:
- Node.js (версия 14 или выше)
- npm или yarn

### Установка

1. **Клонируйте репозиторий или перейдите в папку проекта:**
```bash
cd TodoApp
```

2. **Установите зависимости:**
```bash
npm install
```

3. **Запустите приложение:**
```bash
npm start
```

Приложение откроется в браузере по адресу [http://localhost:3000](http://localhost:3000)

## 📖 Как Использовать

### Добавление Задачи
1. Введите название задачи в поле "Введите название задачи..."
2. (Опционально) Добавьте описание в текстовую область
3. Нажмите кнопку "➕ Добавить задачу"

### Отметить Задачу как Выполненную
- Нажмите на чекбокс слева от задачи или кнопку "○" справа

### Удалить Задачу
- Нажмите на кнопку "🗑️" рядом с задачей

### Фильтрация
- **Все** - Показать все задачи
- **Активные** - Показать только невыполненные задачи
- **Выполненные** - Показать только выполненные задачи

## 💡 Функциональность в Деталях

### App Component (Главный компонент)
- Управляет состоянием всех задач
- Обрабатывает добавление, удаление и переключение статуса
- Управляет фильтрацией и отображением статистики
- Синхронизирует данные с localStorage

### TodoForm Component (Форма добавления)
- Валидация входных данных
- Ограничение на длину названия (100 символов) и описания (500 символов)
- Счетчик оставшихся символов
- Обратная связь пользователю (ошибки и успешное добавление)

### TodoList Component (Список задач)
- Отображает отфильтрованный список задач
- Передает функции управления дочерним компонентам

### TodoItem Component (Отдельная задача)
- Отображает информацию о задаче
- Функции переключения and удаления с анимацией
- Отображает дату создания в локальном формате

## 🎨 Дизайн и Анимации

### Цветовая Схема
- **Основной цвет** - Indigo (#6366f1) для основных элементов

- **slideInUp** - Плавное появление элементов списка

- **bounceIn** - Отскок при загрузке формы

- **Desktop** (1200px и выше)
- **Tablet** (768px - 1199px)
- **Mobile** (480px - 767px)
- **Small Mobile** (до 480px)

## 💾 Сохранение Данных

Все задачи автоматически сохраняются в **localStorage** браузера. Это означает, что ваши задачи будут сохранены даже после:
- Закрытия вкладки браузера
- Закрытия браузера
- Перезагрузки страницы

Данные хранятся в формате JSON под ключом `todos`.
